Teacher Dashboard Plugin Test

Usage Instructions:

  1. Add the shortcode [teacher_dashboard] to any WordPress page or post
  2. The plugin will automatically detect if the user has teacher roles:
    • school_teacher
    • instructor
    • Instructor
    • wdm_instructor
  3. If the user is a teacher, it will show their groups
  4. Click on any group to see the students in that group

Features:

  • ✅ Proper user role detection
  • ✅ Stable group retrieval using proven SQL queries
  • ✅ Interactive group selection
  • ✅ Student list display
  • ✅ Responsive design
  • ✅ Clean, modern UI

Shortcode:

[teacher_dashboard]
